# Research Workflow (Evidence-first)

Usage: `Apply workflow: .clinerules/workflows/research.md`

- [ ] Define the exact question and required confidence level.
- [ ] Use `context-7` first for library/framework documentation questions.
- [ ] Use `web-search` for external or time-sensitive claims; collect 3-5 quality sources.
- [ ] Extract only relevant sections from each source (no bulk page ingestion).
- [ ] Record concrete facts with source URLs and publication/update dates.
- [ ] Mark uncertain items explicitly instead of inferring unsupported details.
- [ ] Produce a compact conclusion with citations and open questions.
